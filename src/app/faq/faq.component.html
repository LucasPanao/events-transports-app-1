O que é Angular?

Angular é uma plataforma e framework para construção de aplicações web em HTML e TypeScript. É desenvolvida pela equipe do Angular no Google e por uma comunidade de indivíduos e corporações.

 Como instalo o Angular?

Você pode instalar o Angular usando o Angular CLI. Execute o comando para instalar o Angular CLI globalmente.

 Quais são as principais características do Angular?

Algumas das principais características do Angular incluem a injeção de dependência, ferramentas poderosas de template, roteamento, suporte para formulários reativos e muito mais.
